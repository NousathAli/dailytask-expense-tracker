# DailyTask Expense Tracker App

DailyTask Expense Tracker is a mobile/web application for managing daily task updates, petty cash entries, reimbursement requests, settlement reports, and admin review workflow.

## Project Purpose

This app was designed to reduce manual tracking through WhatsApp, Excel, and paper-based expense records. It provides a structured workflow for staff users and admin users.

## Key Features

- Daily task submission
- Petty cash entry
- Petty cash month closing
- Reimbursement request tracking
- Settlement report view
- Branch/entity selection
- Admin report page
- Mobile-friendly dashboard

## Tech Stack

- React Native / Expo
- JavaScript
- Firebase / Firestore
- Mobile-first UI design

## User Roles

### Staff User
- Add daily task updates
- Submit petty cash expenses
- Submit reimbursement claims
- View settlement status

### Admin / Team Lead
- Review submitted entries
- Filter reports
- View settlement details
- Track pending and completed entries
